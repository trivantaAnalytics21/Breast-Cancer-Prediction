# Breast Cancer Prediction System

A Machine Learning-based web application that predicts whether a breast tumor is Benign or Malignant using medical diagnostic features and trained classification models.

---

## Features

- Predicts Breast Cancer Diagnosis
- Real-time Prediction Results
- User-friendly Web Interface
- Data Preprocessing & Feature Scaling
- Machine Learning-based Classification
- Responsive and Modern UI
- Fast and Accurate Predictions

---

## Tech Stack

### Frontend
- HTML
- CSS
- Bootstrap

### Backend
- Python
- Flask

### Machine Learning & Data Processing
- Scikit-learn
- Pandas
- NumPy

### Model Development
- Logistic Regression
- Data Scaling
- Model Training & Evaluation

---

## How It Works

1. User enters medical diagnostic values
2. Input data preprocessing
3. Feature scaling applied
4. Trained ML model analyzes data
5. Prediction generated:
   - Benign
   - Malignant
6. Result displayed instantly on screen
